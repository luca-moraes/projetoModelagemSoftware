Operação já efetuada

UMLUseCase
Untitled :: Model :: Operação já efetuada

Description

Fluxo principal:

  1. Usuário tenta cancelar ou modificar a ordem.

  2. O status da ordem é checado na B3.

  3. O status informa que a ordem já foi efetuada, portanto a alteração não é permitida.

  4. A informação é retornada pela B3 para a corretora.

Fluxo alternativo:

  1. Não foi possível realizar o contato com a B3, um erro é retornado ao invés da informação de status da ordem.

Caso de uso referentes aos requisitos :

  • 9 e 10.
