Recuperar senha

UMLUseCase
Untitled :: Model :: Recuperar senha

Description

Fluxo principal:

  1. Usuário identifica-se com com a informação de login.

  2. Usuário solicita recuperação de senha por e-mail.

  3. Corretora envia mensagem com o link de recuperação de senha para o e-mail pessoal do usuário.

  4. Usuário após clicar no link é enviado a uma página de recuperação, digita e confirma uma nova senha.

Fluxo alternativo:

  1. Usuário utiliza o link do e-mail após o tempo 't' de validade, a alteração de senha é negada e um novo e-mail deve ser solicitado.

Caso de uso referentes aos requisitos :
