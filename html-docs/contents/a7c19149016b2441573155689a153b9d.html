Enviar ordem de compra

UMLUseCase
Untitled :: Model :: Enviar ordem de compra

Description

Fluxo principal:

  1. Usuário acessa a área de novas ordens de compra de ações.

  2. Usuário escolhe a ação desejada para efetuar uma nova ordem de compra, podendo utilizar filtros para as opções exibidas.

  3. Usuário escolhe o tipo de ordem de compra, conforme os casos de uso extends.

  4. Usuário preenche e confirma as informações da OC.

  5. Ordem de compra é postada pela corretora ao enviar para a B3.

  6. Após o Include caso de uso "Postar ordem de compra", a B3 retorna a confirmação de que a ordem de compra foi postada.

Fluxo alternativo:

  1. Include caso de uso "Saldo insuficiente", ele recebe um aviso para depositar um valor maior em sua conta.

  2. Include caso de uso "Saldo insuficiente", a operação não é realizada por erro na confirmação do saldo, o usuário é avisado para tentar novamente mais tarde.

  3. Include caso de uso "Postar ordem de compra" retorna que houve um erro de comunicação na postagem da ordem, portanto o cliente deve tentar novamente mais tarde.

Caso de uso referentes aos requisitos :

  • 8, 12, 14 e 16.
