Consultar informações de ordem de venda

UMLUseCase
Untitled :: Model :: Consultar informações de ordem de venda

Description

Fluxo principal:

  1. Usuário acessa a área de ordens de venda postadas.

  2. Usuário pode entrar na opção de ordens não efetuadas para visualizar status de ordens abertas, ou ordens efetuadas para consultar seu histórico de vendas.

  3. Usuário escolhe a ordem de venda desejada, podendo utilizar filtros para as opções exibidas.

  4. Usuário visualiza as informações adicionais daquela ordem.

Fluxo alternativo:

  1. Nenhuma informação é exibida pois o usuário ainda não enviou alguma ordem para a corretora.

Caso de uso referentes aos requisitos :

  • 20, 22 e 24.
