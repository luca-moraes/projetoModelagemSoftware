Cadastro

UMLUseCase
Untitled :: Model :: Cadastro

Description

Fluxo principal:

  1. Usuário registra suas informações pessoais.

  2. Usuário envia documentação solicitada pela corretora.

  3. Aguarda cadastro ser aprovado pelo gerente da corretora, Include caso de uso "Validar Cadastro".

  4. Após aprovação cadastro cadastro é liberado e usuário é notificado do processo.

Fluxo alternativo:

  1. Include caso de uso "Validar Cadastro", o cadastro foi negado, portanto, usuário deve corrigir as informações solicitadas para submeter uma tentativa de cadastro novamente ou cancelar a solicitação de cadastro.

Caso de uso referentes aos requisitos :

  • 1 e 4.
