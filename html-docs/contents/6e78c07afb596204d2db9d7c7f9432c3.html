Visualizar carteira de ações

UMLUseCase
Untitled :: Model :: Visualizar carteira de ações

Description

Fluxo principal:

  1. Usuário entra na tela de carteira de ações, podendo alterar entre as abas de lista de ações e grupos de ações, esta mostra as ações por grupos criados pelo usuário e aquela mostra uma lista com todas as ações adquiridas pelo cliente, a cotação das ações é exibida ao lado de cada uma delas, através de ícones interativos o usuário pode personalizar a visualização das cotações.

  2. Usuário clica em uma ação específica para exibir mais detalhes e escolhe a opção de "Visualizar Extrato"

  3. Usuário pode fazer um negócio rápido através do include caso de uso "Negócios rápidos em carteira".

Fluxo alternativo:

  1. Nenhuma informação é exibida pois o usuário ainda não possui ações na corretora.

Caso de uso referentes aos requisitos :

  • 26, 33, 34 e 35.
