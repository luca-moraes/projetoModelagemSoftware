Visualizar carteira de ações

UMLUseCase
Untitled :: Model :: Visualizar carteira de ações

Description

Fluxo principal:

  1. Usuário entra na tela de carteira de ações, podendo alterar entre as abas de lista de ações e grupos de ações, esta mostra as ações por grupos criados pelo usuário e aquela mostra uma lista com todas as ações adquiridas pelo cliente.

Fluxo alternativo:

  1. Nenhuma informação é exibida pois o usuário ainda não possui ações na corretora.
