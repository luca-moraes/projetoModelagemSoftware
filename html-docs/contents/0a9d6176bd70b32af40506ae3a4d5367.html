Modificar ordem de compra

UMLUseCase
Untitled :: Model :: Modificar ordem de compra

Description

Fluxo principal:

  1. Usuário acessa a área de ordens de compra postadas e entra na opção de ordens não efetuadas.

  2. Usuário escolhe a ordem de compra desejada, podendo utilizar filtros para as opções exibidas e visualizar apenas OC relacionadas a ação selecionada.

  3. Usuário escolhe e confirma a opção de modificar OC.

  4. Usuário ajustar as informações da OC e confirma modificação.

  5. Corretora envia as modificações da OC para a B3, que confirma a mudança.

Fluxo alternativo:

  1. Include caso de uso "Operação já efetuada", o usuário é avisado que a ordem não pode ser cancelada.

  2. A operação não é efetuada por erro no Include caso de uso "Operação já efetuada", o cliente é informado do erro e solicita-se que ele tente realizar a operação novamente mais tarde.

Caso de uso referentes aos requisitos :

  • 10 e 15.
