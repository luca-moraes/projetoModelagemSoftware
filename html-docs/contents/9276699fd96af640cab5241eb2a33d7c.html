Diagrama Classes

UMLUseCaseSubject
Untitled :: Model :: Diagrama Classes

Description

Fluxo principal:

  1. Usuário entra na tela de carteira de ações, no topo das abas de opções de visualização, ele utiliza a opção de criar grupo de ações.

  2. Uma tela aparece com um campo para preencher as informações como o nome do grupo.

  3. O usuário preenche as informações e salva o novo grupo criado.

Fluxo alternativo:

  1. O grupo não é criado pois o nome inserido já está em uso, o cliente é avisado para alterar o nome do grupo.
