Modificar ordem de venda

UMLUseCase
Untitled :: Model :: Modificar ordem de venda

Description

Fluxo principal:

  1. Usuário acessa a área de ordens de venda postadas e entra na opção de ordens não efetuadas.

  2. Usuário escolhe a ordem de venda desejada, podendo utilizar filtros para as opções exibidas e visualizar apenas OV relacionadas a ação selecionada.

  3. Usuário escolhe e confirma a opção de modificar OV.

  4. Usuário ajustar as informações da OV e confirma modificação.

  5. Corretora envia as modificações da OV para a B3, que confirma a mudança.

Fluxo alternativo:

  1. Include caso de uso "Operação já efetuada", o usuário é avisado que a ordem não pode ser cancelada.

  2. A operação não é efetuada por erro no Include caso de uso "Operação já efetuada", o cliente é informado do erro e solicita-se que ele tente realizar a operação novamente mais tarde.

Caso de uso referentes aos requisitos :

  • 19 e 24.
