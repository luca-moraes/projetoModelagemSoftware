Criar alertas de cotação

UMLUseCase
Untitled :: Model :: Criar alertas de cotação

Description

Fluxo principal:

  1. Usuário ao logar na plataforma clica no ícone de "!" dentro de um triângulo (para simbolizar o alerta), no canto superior da tela.

  2. Usuário é levado para a página de criação de alertas de cotação.

  3. Usuário configura um alerta para uma ação específica com as informações necessárias.

  4. Usuário confirma a criação do alerta.
