Validar Cadastro

UMLUseCase
Untitled :: Model :: Validar Cadastro

Description

Fluxo principal:

  1. Gerente acessa a área de solicitações de cadastro.

  2. Gerente revisa as informações de cadastro cadastro e os documentos.

  3. Gerente aprova o cadastro e libera acesso para o novo cliente.

Fluxo alternativo:

  1. As informações registradas e os documentos enviados não conferem, portanto, cadastro é negado e o usuário é notificado do processo.

Requisitos referentes ao caso de uso:

  • 1 e 4.
