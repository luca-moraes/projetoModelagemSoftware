Negócios rápidos com ações em alta ou baixa

UMLUseCase
Untitled :: Model :: Negócios rápidos com ações em alta ou baixa

Description

Fluxo principal:

  1. Dentro das visualizações de maiores altas e baixas o usuário pode fazer negócios rápidos clicando na opção de "+" ou "-" ao lado de cada ação.

  2. Usuário confirma o negócio rápido clicando no símbolo de "ok" ao lado dos ouros botões.

  3. Negócio rápido é efetuado.

Fluxo alternativo:

  1. Include caso de uso "Saldo insuficiente", ele recebe um aviso para depositar um valor maior em sua conta.

  2. Por um erro de comunicação a operação não é realizada e cliente é notificado a tentar novamente mais tarde.
