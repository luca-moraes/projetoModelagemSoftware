Visualizar maiores altas e baixas

UMLUseCase
Untitled :: Model :: Visualizar maiores altas e baixas

Description

Fluxo principal:

  1. Usuário após escolher a opção de visualizar maiores altas e baixas, recebe as duas listas para cada tipo com as principais ações.

  2. Usuário pode clicar para visualizar o gráfico de uma ação específica e ajustar a visualização para barras de candle conforme o include caso de uso "Visualizar em candles".

  3. Usuário pode fazer um negócio rápido através do include caso de uso "Negócios rápidos com ações em alta ou baixa".
