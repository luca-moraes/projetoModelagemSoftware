Saldo insuficiente

UMLUseCase
Untitled :: Model :: Saldo insuficiente

Description

Fluxo principal:

  1. Antes de uma ordem ser confirmada o saldo disponível do usuário é checado.

  2. O saldo é insuficiente para a ordem ser executada.

  3. O usuário é notificado para depositar um valor maior caso queira executar a ordem.

Fluxo alternativo:

  1. Por algum erro o saldo não pode ser confirmado, a operação é cancelada e o usuário é notificado para tentar novamente mais tarde.

Caso de uso referentes aos requisitos :

  • 8, 12, 14 e 16.
