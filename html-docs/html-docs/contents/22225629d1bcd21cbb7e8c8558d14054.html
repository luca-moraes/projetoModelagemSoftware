Aviso de bloqueio

UMLUseCase
Untitled :: Model :: Aviso de bloqueio

Description

Fluxo principal:

  1. Quando o acesso do usuário é bloqueado por excesso de tentativas um alerta é enviado por SMS e e-mail.

  2. A data e hora do bloqueio e do último login válido são enviados juntos para aviso do usuário.
