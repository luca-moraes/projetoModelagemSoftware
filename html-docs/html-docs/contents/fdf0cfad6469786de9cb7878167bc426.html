Agrupar ações da carteira

UMLUseCase
Untitled :: Model :: Agrupar ações da carteira

Description

Fluxo principal:

  1. Usuário entra na tela de carteira de ações, na aba de lista de ações .

  2. Ao lado de cada ação listada o usuário seleciona o ícone da opção para adicionar ação a um grupo.

  3. A lista de grupos disponíveis aparece ao lado da ação.

  4. O usuário escolhe o grupo desejado e ação é adicionada.

Fluxo alternativo:

  1. A ação não é adicionada ao grupo pois ela já faz parte dele.
