Checar status de ordem

UMLUseCase
Untitled :: Model :: Checar status de ordem

Description

Fluxo principal:

  1. Recebe as informações referentes a ordem que deve ser checada.

  2. Verifica as informações da ordem.

  3. Retorna as informações de status da ordem solicitadas para a corretora.

Fluxo alternativo:

  1. As informações da ordem não puderam ser verificas ou houve algum erro de comunicação, um erro é retornado para a solicitação de checagem.

Caso de uso referente aos requisitos:

  • 9 e 10.
