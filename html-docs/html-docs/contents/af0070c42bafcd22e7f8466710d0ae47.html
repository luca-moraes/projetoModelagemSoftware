Description

Fluxo principal:

  1. Usuário clica na aba com as opções de simulação.

  2. Usuário recebe créditos fictícios para simular opções de negócio.

  3. Usuário simula a compra ou venda de opções e pode acompanhar os resultados na mesma aba.

  4. Usuário confirma a criação do alerta.
