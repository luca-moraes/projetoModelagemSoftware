Enviar ordem de venda

UMLUseCase
Untitled :: Model :: Enviar ordem de venda

Description

Fluxo principal:

  1. Usuário acessa a área de novas ordens de venda de ações.

  2. Usuário escolhe a ação desejada para efetuar uma nova ordem de venda, podendo utilizar filtros para as opções exibidas.

  3. Usuário escolhe o tipo de ordem de venda, conforme os casos de uso extends.

  4. Usuário preenche e confirma as informações da OV.

  5. Ordem de venda é postada pela corretora.

Fluxo alternativo:

  1. Include caso de uso "Postar ordem de venda" retorna que houve um erro de comunicação na postagem da ordem, portanto o cliente deve tentar novamente mais tarde.

Caso de uso referentes aos requisitos :

  • 17, 21, 23 e 25.
